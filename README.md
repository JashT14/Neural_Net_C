# Neural_Net_C
A simple project made in C, understanding the basics for Neural Networks. One can refer this to understand Neural Network in C.

### Core Concepts & Mathematics

* **Matrix Calculus:** Weights ($W$) and biases ($b$) stored as matrices; layer transitions computed via $Wx + b$.
* **Sigmoid Activation:** Non-linear squashing function: $\sigma(x) = \frac{1}{1 + e^{-x}}$.
* **Forward Propagation:** Sequential data flow through layers.
* **Mean Squared Error (MSE):** Cost function to quantify prediction error.
* **Finite Difference:** Numerical approximation of derivatives by perturbing weights to calculate gradients.
* **Gradient Descent:** Parameter optimization using a learning rate to minimize cost.
